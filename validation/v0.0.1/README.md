# v0.0.1 Physical Validation Evidence

Only sanitized, schema-validated evidence belongs here. Raw evidence must never enter Git. Each run uses an alias such as `rig01` and is linked to one authoritative RC binary SHA-256 and one encrypted private archive ciphertext SHA-256.

No run is accepted until every mandatory check passes and the external `tar.zst.age` archive has been independently decrypted and verified.
