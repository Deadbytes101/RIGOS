# Historical Notice: Pre-Rename RC1 Is Obsolete

The pre-release source candidate built under the former `DBYTE RIGOS`, `dbyte-rigos`, and `dbyte.rigos.*` names is obsolete. It was never a final release and must not be used to tag `v0.0.1` or supply acceptance evidence.

Its commits remain in history for auditability. The first eligible renamed candidate is `v0.0.1-rc2`, built from a new source bundle and validated from scratch.
