# Miner Provenance

```text
MINER:                  official XMRig 6.26.0
MODIFIED BY RIGOS:      no
RIGOS FEE:              0%
UPSTREAM DONATION:      disclosed and unchanged
RIGOS RECEIVES IT:      no
OFFLINE AVAILABILITY:   bundled in immutable image
```

Inspect locally with `rigosctl miner provenance`, `rigosctl about`, or their
`--json` forms. The archive hash and embedded binary hash are distinct fields.
A future patched miner requires a separate build identity, corresponding source,
patch set, reproducible build, GPL review, and old-CPU validation.
