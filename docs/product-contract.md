# RIGOS Product Contract

RIGOS is a CPU-only, pool-neutral, USB-native, local-first Linux mining operating system.

```text
OS LICENSE COST:   0
WORKER LIMIT:      NONE
MONTHLY FEE:       0
CLOUD FEE:         0
RIGOS DEV FEE:     0
ACCOUNT REQUIRED:  NO
ACTIVATION:        NO
FORCED POOL:       NO
```

Electricity, Internet access, hardware, pool fees, and third-party miner fees remain external costs outside RIGOS control. The codebase contains no billing, entitlement, activation, trial, balance, payment-dependent miner control, or remote kill-switch runtime.

RIGOS operates without RIGOS infrastructure. Internet access supplies pool work and optional market/update data; it does not own the node, authorize mining, or impose worker limits.

The machine-readable source of truth is [`configs/product-contract.toml`](../configs/product-contract.toml), verified by `rigos-contract` tests.

Claims of superiority over other products are forbidden until comparative physical evidence exists. Current performance numbers are acceptance targets, not marketing facts.
