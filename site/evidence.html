What this proves

  • The Alpha.25 source booted and reached ready state on recorded real hardware.
  • Persistent state, runtime activation, network readiness, exact thread configuration, huge pages and miner ownership aligned on the sampled nodes.
  • Configured mining produced accepted shares with no rejected shares in the captured snapshots.
  • The short operator command reported sanitized status without dumping private runtime configuration.

What this does not prove

  • Compatibility with arbitrary x86_64 machines.
  • Repeated cold-boot and power-loss reliability.
  • Correct recovery through sustained network or pool failure.
  • Long unattended soak behavior.
  • Guaranteed hashrate on different CPUs, memory, firmware or thermal conditions.
A physical sample is evidence for that node under those conditions. It does not justify a broader claim.
